The program first reads  hello.txt file and displays the text that was inside the file.
Then you can enter chars and you can also delete  chars with the Backspace button.
You move the cursor only with the left and right arrows.
When you are done playing around with the text, press enter to save the text,it will write the text back to hello.txt file, and it ends the program.
You can also end the program using  the Esc button.
